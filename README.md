#  What's this?

[ippp.gq](https://ippp.gq) is a simple and straightforward website to display ipv4 address information




## Installation

Clone the repo
```bash
git clone https://github.com/TheRedstoneRadiant/ippp.gq
cd ippp.gq
```

Install dependencies
```bash
npm install
```

Run server
```bash
npm start
````
